# Dmitry Dronov - Web Products & Landing Pages

Этот проект представляет собой профессиональный лендинг-пейдж для веб-разработчика Дмитрия Дронова, специализирующегося на создании конверсионных лендингов и веб-продуктов.

## 🚀 Рабочая страница

**Живая версия доступна по адресу:** https://dronov-dmitry.github.io/

## 📋 О проекте

Лендинг-пейдж разработан с фокусом на конверсию и современную эстетику. Основные особенности:

- **Mobile First дизайн** - адаптивная вёрстка, оптимизированная для мобильных устройств
- **Многоязычная поддержка** - 5 языков: русский, немецкий, английский, французский, испанский
- **Автоматическое определение языка** - язык выбирается на основе настроек браузера пользователя
- **Калькулятор проектов** - интерактивный инструмент для оценки стоимости разработки
- **Портфолио проектов** - демонстрация реализованных работ с ссылками на GitHub и живые версии

## 🛠 Технологии

- Чистый HTML, CSS, JavaScript
- Адаптивная вёрстка (CSS Grid, Flexbox)
- Современный дизайн с тёмной темой
- Локализация через JavaScript
- Форма с валидацией и отправкой email

## 🌐 Локализация

Страница автоматически определяет язык браузера пользователя и переключается на один из поддерживаемых языков:
- Русский (ru)
- Немецкий (de) 
- Английский (en) - язык по умолчанию
- Французский (fr)
- Испанский (es)

Пользователь также может вручную выбрать язык через выпадающий список в верхней панели.
